# 🌟 Smooth Scrolling Website Project 🌟
The Smooth Scrolling Website Project is a web application that provides a seamless and smooth scrolling experience for users. The project is built using HTML, CSS, and JavaScript, and it utilizes the browser's built-in `document` and `window` objects, as well as the `requestAnimationFrame` function, to achieve the smooth scrolling effect. The project's core feature is the smooth scrolling animation, which is triggered by clicking on navigation links with the class `js-scroll`. The project also includes a basic structure and content, including a navigation menu, hero section, and other sections.

## 🚀 Key Features
* Smooth scrolling animation triggered by clicking on navigation links with the class `js-scroll`
* Basic structure and content, including a navigation menu, hero section, and other sections
* Integration with external stylesheet (`style.css`) for styling and layout
* Use of browser's built-in `document` and `window` objects, as well as the `requestAnimationFrame` function

## 🛠️ Tech Stack
* Frontend: HTML, CSS, JavaScript
